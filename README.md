# Environment Configuration Manager

A DevOps-focused application demonstrating configuration management across environments.

## Project Structure
- **backend/**: Express API for config management
- **frontend/**: UI for environment configuration
- **docker/**: Dockerfile and Docker configuration
- **k8s/**: Kubernetes manifests (Deployment, Service, ConfigMap)
- **.github/workflows/**: CI/CD pipeline
